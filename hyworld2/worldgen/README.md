# World Generation Module

This module provides world generation capabilities for HY-World 2.0. 

## Status

⚠️ **Note**: This module is currently not open-sourced in this repository.

## External Reference

The latest preview version has been released on the original [WorldStereo](https://github.com/FuchengSu/WorldStereo) project, which provides world generation functionality that this module will work.

When available, this module will contain:
- Configuration files for integrating with the external world generation project
- Interface definitions and API documentation
- Usage examples and integration guides

## Future Plans

We plan to open-source the complete world generation pipeline in the future. Please check back for updates.

For more information about HY-World 2.0's world generation capabilities, please refer to the main [README.md](../README.md).
